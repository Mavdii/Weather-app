### Required API Keys

To use all features, you'll need to set up the following:

1. **Weather API Key** - Get from your weather data provider
2. **Newell AI API Key** - Get from [Newell AI](https://newell.ai) for AI insights

### Environment Setup

The app uses the following environment variables (already configured in the release build):

```env
EXPO_PUBLIC_NEWELL_API_KEY=your_key_here
EXPO_PUBLIC_NEWELL_API_URL=https://api.newell.ai
EXPO_PUBLIC_WEATHER_API_KEY=your_key_here
```

## ğŸ“– Usage Guide

### Getting Started
1. **Launch the app** - Allow location permissions when prompted
2. **View weather** - See current weather for your location
3. **Search cities** - Tap the search icon to find cities worldwide
4. **Add favorites** - Tap the heart icon on any city to save it
5. **Explore maps** - Navigate to the Map tab for weather visualization
6. **Customize** - Visit Settings to adjust preferences

### Pro Tips
- **Long-press** on favorites to remove them
- **Pull to refresh** to update weather data
- **Swipe** through hourly forecasts
- Enable **notifications** for weather alerts
